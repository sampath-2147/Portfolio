# Sampath  Portfolio (Tailwind + Purple/White)

## Run locally (VS Code / VS Code Insiders)
1. Open this folder in VS Code
2. Install extension: **Live Server**
3. Right-click `index.html` -> **Open with Live Server**

## Deploy to GitHub Pages (optional)
- Push this repo to GitHub
- Go to Settings -> Pages -> Deploy from branch

## Deploy to Vercel (recommended)
1. Create a GitHub repo and push this code
2. Go to Vercel -> New Project -> Import from GitHub
3. Deploy (Vercel auto-deploys on every push)

## Files
- `index.html` Home (profile popup + typing animation + resume button)
- `projects.html` Projects (3D cards + modal)
- `experience.html` Experience (company card + modal + link)
- `education.html` Education (cards + modal)
- `contact.html` Contact (3D flip cards)
- `flowchart.html` Resume flowchart
- `assets/` images + resume PDF
